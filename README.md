## 🚀 快速开始（3步）

### 第1步：准备文件

1. **准备照片**（16张）
   - 1张封面照片（重命名为 `cover.jpg`）
   - 15张内页照片（重命名为 `photo1.jpg ~ photo15.jpg`）
   - 放入 `images` 文件夹

2. **准备音乐**（1个）
   - 选择一首喜欢的音乐
   - 转换为MP3格式
   - 重命名为 `bg-music.mp3`
   - 放入 `audio` 文件夹

### 第2步：压缩照片 ⚠️ 重要！

**为什么要压缩**：
- 照片太大会导致加载慢或失败
- 建议每张不超过500KB

**如何压缩**：
1. 双击打开 `图片压缩工具.html`
2. 拖拽所有照片到工具中
3. 设置质量为80-85%
4. 下载压缩后的照片
5. 替换到 `images` 文件夹

### 第3步：修改文字

1. 打开 `index.html`
2. 搜索 `【文本替换区】`
3. 修改引号内的文字为您的情话
4. 保存文件

## ⚠️ 常见问题解决

### ❌ 问题：图片或音乐加载不出来

**原因**：文件太大或路径错误

**解决**：
1. 📖 查看 `一键解决方案.md`（最快）
2. 🔧 使用 `图片压缩工具.html` 压缩照片
3. 🔍 使用 `诊断工具.html` 检查问题
4. 📋 查看 `常见问题解决方案.md`

---

### ❌ 问题：音乐不自动播放

**原因**：浏览器限制自动播放

**解决**：
- 点击页面任意位置
- 点击右上角音乐按钮

---

### ❌ 问题：手机上显示不正常

**原因**：照片太大，手机加载慢

**解决**：
- 照片压缩到300KB以下
- 使用4G/5G网络测试
- 等待更长加载时间

## ⚡ 快速问题排查

### 遇到问题？按顺序尝试：

1. **照片不显示？**
   ```
   → 打开"图片压缩工具.html"
   → 压缩所有照片
   → 替换到images文件夹
   → 按Ctrl+F5刷新
   ```

2. **音乐不播放？**
   ```
   → 检查文件是否为bg-music.mp3
   → 点击页面任意位置
   → 点击右上角音乐按钮
   ```

3. **部署后打不开？**
   ```
   → 检查所有文件名是否为英文小写
   → 确认所有文件已上传
   → 清除浏览器缓存
   → 等待几分钟后重试
   ```

4. **还是不行？**
   ```
   → 打开"诊断工具.html"
   → 查看具体错误
   → 阅读"常见问题解决方案.md"
   ```
